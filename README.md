# 🎓 MLN QuizMaster - Hệ Thống Ôn Tập Trắc Nghiệm Thông Minh

**MLN QuizMaster** là một ứng dụng web (Single Page Application) hiện đại giúp ôn tập kiến thức và tự luyện trắc nghiệm hiệu quả qua:

- **Thẻ ghi nhớ (Flashcards)**: Học bài qua các thẻ lật tương tác, hỗ trợ phát âm đọc câu hỏi tự động.
- **Học dạng danh sách (List View)**: Đọc nhanh toàn bộ câu hỏi và đáp án giải thích đi kèm.
- **Luyện trắc nghiệm (Quiz)**: Làm bài thi thử trắc nghiệm ngẫu nhiên với chức năng xáo trộn câu hỏi/đáp án, tự động chấm điểm và tổng hợp các câu trả lời sai để ôn tập lại.
- **Quản lý câu hỏi khó (Starred)**: Đánh dấu sao lưu riêng các câu hỏi khó để thuận tiện cho việc học tập trung.
- **Hệ thống đăng nhập (MockAPI Auth)**: Hỗ trợ xác thực đăng nhập tài khoản học viên và duy trì trạng thái hoạt động tiện lợi.
